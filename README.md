# Philosophy Explorer: Philosophical Traditions Comparison

## Overview

Philosophy Explorer is a comparative web resource to help learners explore and understand the similarities and unique features of three great philosophical traditions: Western, Asian, and Indian. The site presents:

- Key approaches, methods, and goals of each tradition
- Comparison of answers to universal questions (morality, reality, knowledge, etc.)
- Visual and textual summaries for clarity
- Insights into the modern relevance of ancient wisdom

## Features

- **Comparative tables**: Side-by-side understanding of Western, Asian, and Indian philosophies.
- **Universal questions**: How different cultures answer the big questions of human existence.
- **Key insights**: Universal concerns and complementary approaches across traditions.
- **Contemporary relevance**: Personal growth, global ethics, and responsible leadership lessons.

## Usage

Open `comparison.html` in any web browser. Navigate between home, Western, Asian, and Indian philosophy sections using the menu.

## Educational Purpose

Content is intended for educational use—helping students, educators, and the curious explore philosophical ideas across cultures.
